# LeetCode

## [175. Combine Two Tables](https://leetcode.com/problems/combine-two-tables/)

### MySQL

```sql
SELECT p.firstName, p.lastName, a.city, a.state
FROM Person p
LEFT JOIN Address a
    ON p.personId = a.personId;
```

---

### MSSQL

```sql
SELECT p.firstName, p.lastName, a.city, a.state
FROM Person p
LEFT JOIN Address a
    ON p.personId = a.personId;
```
