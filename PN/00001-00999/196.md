# LeetCode

## [196. Delete Duplicate Emails](https://leetcode.com/problems/delete-duplicate-emails/)

### MySQL

```sql
DELETE p1
FROM Person p1
JOIN Person p2
  ON p1.email = p2.email
 AND p1.id > p2.id;
```

---

### MSSQL

```sql
DELETE FROM Person
WHERE id NOT IN (
    SELECT MIN(id)
    FROM Person
    GROUP BY email
);
```
