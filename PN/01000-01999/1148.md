# LeetCode

## [1148. Article Views I](https://leetcode.com/problems/article-views-i/)

### MySQL

```sql
SELECT DISTINCT viewer_id as id 
FROM Views 
WHERE viewer_id = author_id Order BY viewer_id ASC;
```
