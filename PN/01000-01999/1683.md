# LeetCode SQL

## [1683. Invalid Tweets](https://leetcode.com/problems/invalid-tweets/description/?envType=study-plan-v2&envId=top-sql-50)

### MySQL

```sql
SELECT tweet_id 
FROM Tweets 
WHERE LENGTH(content) > 15;
```

---

### MSSQL

```sql
SELECT tweet_id 
FROM Tweets 
WHERE LEN(content) > 15;
```
