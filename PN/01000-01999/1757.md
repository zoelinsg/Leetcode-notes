# LeetCode SQL

## [1757. Recyclable and Low Fat Products](https://leetcode.com/problems/recyclable-and-low-fat-products/description/?envType=study-plan-v2&envId=top-sql-50)

### MySQL

```sql
SELECT product_id 
FROM Products 
WHERE low_fats = 'Y' and recyclable = 'Y';
```

---

### MSSQL

```sql
SELECT product_id 
FROM Products 
WHERE low_fats = 'Y' and recyclable = 'Y';
```
